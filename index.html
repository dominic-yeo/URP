Very Simple Guide: Build a PHP Application

1. Install PHP

Install PHP on your machine. Verify with:

php --version

2. Create project files

Make a folder and create a single PHP file:

mkdir hello-php
    cd hello-php
    echo '<?php echo "Hello, world!"; ?>' > index.php

3. Run with the built-in server

Start the PHP development server and open the page in your browser:

php -S localhost:8000
    # then visit http://localhost:8000/ in your browser

4. Example: simple dynamic greeting

Replace `index.php` with this to accept a `name` query parameter:

<?php
    $name = isset($_GET['name']) ? htmlspecialchars($_GET['name']) : 'Developer';
    echo "Hello, {$name}!";
    ?>

5. Next steps (optional)

  • Learn routing and templates (e.g., use a microframework or simple includes).
  • Use environment variables for configuration (do not hardcode secrets).
  • Set up version control with Git and add a basic README.
